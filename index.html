Todo Tray App

A lightweight system tray task manager built with Electron. Quick access to your tasks without desktop clutter or workflow interruptions.

See Demo Tech Details

Key Features

Built for productivity and minimal distraction

🖱️

System Tray Integration

Lives quietly in your system tray. Single click access with intelligent positioning near the tray icon.

⚡

Global Shortcuts

Ctrl+Shift+T toggles the window from anywhere. Add tasks instantly without switching applications.

🌙

Dark & Light Themes

Clean themes with smooth transitions. Consistent design language across both light and dark modes.

📊

Smart Task Filtering

Active and completed task views. Recent completed tasks (24h) to keep focus on current priorities.

💾

Local Data Storage

Tasks saved to local JSON file. No cloud dependency means your data stays private and works offline.

🎯

Distraction-Free Design

Auto-hide on focus loss, minimal interface, zero notifications. Enhances productivity without interruption.
